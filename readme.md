### USBシリアルのアクセス権限の設定。
mode666で読み込むようにする。
```bash
sudo nano /etc/udev/rules.d/65-serial.rules
```

```bash
# /etc/udev/rules.d/65-serial.rules
ACTION=="remove", GOTO="serial_end"
SUBSYSTEM!="tty", GOTO="serial_end"
KERNEL=="ttyUSB[0-9]*", MODE="0666"
LABEL="serial_end"
```

```bash
sudo udevadm control --reload
```
